Greencity Website Decision

Stay on Houzez (via agency) vs. Build Custom (Next.js + Admin + Backend)

Goal: Choose the option that best supports growth, performance, integrations, and long‑term ROI.

Learn More Jump to Budget

Current Website Status

Our current website is built on the Houzez WordPress theme ($30): Houzez knowledge base.

Main Problems

  • Performance issues — the website is heavy and slow.
  • Unclear support — if theme support ends, migration will be costly.
  • No SEO — not optimized for search engines.
  • Too much unused functionality — many features we don’t need.
  • Database corruption — data sometimes breaks and needs developer fixes.
  • Unstable functionality — plugin dependencies cause inconsistent behavior (e.g., maps with Sofia).
  • Design limitations — the theme controls layout and features.
  • Limited integrations — restricted and workaround‑heavy (e.g., HubSpot).

Positive Points

  • Cheap and quick to launch.
  • Ready to use “out of the box.”
  • Minimal technical skills required.
  • Some integrations exist, even if limited.

Conclusion

The current website is designed for non‑technical users who need something that “just works” quickly. But once real business logic or complex features are needed — especially after plugin updates — we have to involve developers. That makes maintenance harder and more expensive. And most importantly: performance will never significantly improve with the current setup.

Agency Proposal — Arthur Gorbienko’s Team

They propose rebuilding the website on the same WordPress platform and theme we are using now.

What They Offer

  • A website built on the same platform without bugs (as promised).
  • SEO / AI‑driven SEO.
  • Photo and video production aligned with the design.
  • Design (still theme‑limited) with integrated visuals.
  • Data migration (for an additional fee).
  • Staff training on how to use the site.
  • Ongoing support and maintenance (for an additional fee).
  • Content planning for new pages (location pages, blog, service pages).

Concerns

  • The same fundamental platform limitations may remain.

Positive Points

  • SEO and AI‑driven SEO.
  • New content planning (blogs, location pages, etc.).
  • Photo and video creation for the website.

Our Proposal — Build a Custom Website

We’ll build the site from scratch for clean, fast performance. Inspiration: Toll Brothers.

What We Will Do

  • Build the website from the ground up, using the current site as a data source.
  • Create a unique custom design in Figma.
  • Build a custom admin panel and database.
  • Include only the features we truly need — nothing extra.

Key Benefits

  • High performance: pages load quickly and feel smooth.
  • Native integrations: direct connections with Monday.com, Buildertrend, HubSpot, internal DB, and more.
  • Automations: reduce manual work and human error.
  • Clean and simple: easier to maintain without extra features.

Downsides

  • No photo/video production.
  • No article/blog writing (AI can assist if needed).
  • SEO is shared: we can implement technical SEO, but content and keywords require input.

Summary: Agency is faster but keeps limits. Custom takes longer but gives long‑term performance, scalability, and control. Both need ongoing support, but custom support is easier and more flexible.
